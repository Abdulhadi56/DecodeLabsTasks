# Student Management System

## Project Overview

This project demonstrates Database Integration using PHP and MySQL. The application allows users to perform CRUD (Create, Read, Update, Delete) operations on student records stored in a relational database.

## Features

* Database Connection using MySQL
* Create Student Records
* View Student Records
* Update Student Information
* Delete Student Records
* Data Persistence
* Simple and Clean Interface

## Technologies Used

* PHP
* MySQL
* HTML
* XAMPP

## Database Schema

Table: students

Fields:

* id (Primary Key)
* name
* email
* course

## CRUD Operations

### Create

Add new student records to the database.

### Read

Display all student records.

### Update

Modify existing student information.

### Delete

Remove student records from the database.
